# 🦟 Dengue Prevalence Analysis

This project analyzes the prevalence of Dengue fever using real-world patient datasets. It combines **exploratory data analysis**, **statistical modeling**, and **machine learning** techniques using both **Python** and **R**. The goal is to identify patterns in symptoms, demographics, and blood parameters to support early diagnosis and better understanding of disease spread.
